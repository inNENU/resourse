提示

研究生推免及大家平时常说的保研，为方便大家理解，后文以保研代指。

保研考核内容

  1. 保研考核包含思想品德考核，不合格者不予推荐。

  2. 保研主要考核依据是本科阶段学业综合成绩绩点，不进行考试。

提示

大部分学院进行绩点排名是会选取大类平台课、专业基础课、专业主干课等专业必修课平均学分绩点。

保研途径

保研可以分为普通保研和专项保研两类。

  • 普通保研依据上方考核内容和下方标准进行。

  • 专项保研是一种特殊的保研方式，一般通过教技、外出支教等额外的考核方式和保研条件，适度放宽普通推免的限制条件，以研究生支教团、卓越教师培养计划等形式进行。

专项计划类政策

专项计划类推免生政策将根据国家和学校政策变化做相应调整。

专项计划类条件放宽仍有最低要求。

专项计划类内容

专项计划类考核主要面条件稍差一些的的退伍学生、党员以及学生干部。

其中研究生支教团主要的要求是到指定地点支教一年，以推免教师方向研究生。

学校以前还有 1+3 兼职辅导员的专项计划，及一年兼职辅导员，后三年兼职辅导员同时读研。

保研人数

学校计划总人数，并根据学院 (部) 应届本科毕业生人数，综合考虑学科、基地班等情况，向各学院分配名额。

提示

一般情况下，基地班会拿到更多比例的保研名额，但通常不会拿到全员名额。当然，一般情况下，基地班也会存在部分人员不符合保研条件。

保研条件

  1. 必须是应届本科生

  2. 没有签订不得报考研究生的协议，如公费师范生、优师专项计划生、定向、委托培养生。

  3. 在校期间未受过任何处分。

  4. 完成前三年要求修读必修课程，不及格 (重修及格的视为及格) 或未修的不具备排名资格。

  5. 原则上，大类平台课、专业基础课、专业主干课等专业必修课平均学分绩点在专业中的排名不低于前 30%；

  6. 大学英语四级成绩在 425 分 (含) 以上，其它语种学生大学外语四级成绩需合格 (仅限东北师范大学考点，其它考点无效) ；或东北师范大学英语水平测试成绩达到 70 分(含)以上。

提示

绩点通常只计算大类平台课、专业基础课、专业主干课，也就是大一修读的大类平台下的必修课 (如高数、大物等) 和所有专业必修课。

注意

请特别注意，绩点原则上不能超出 30%。

学校认定的基地班专业学生不受排名限制。

注意

保研对英语提出了要求，要么 CET4 成绩 ≥ 425，要么 DET 分数 ≥ 70。不满足英语成绩要求者，无法进行普通保研。

提示

如果绩点排名超出 30%，或者英语成绩不满足要求，但学生确有优异的科研能力 (即包含下方加分条件的)，如获得院内 2 名以上教授推荐信，可以适当破格，但需公开公示。

注意

签订协议的学生没有保研资格，即使您已经提前违约，缴纳了足额违约金也不行。

保研工作实施

  1. 学院 (部) 会根据学校办法，根据自身专业特点进一步细化并制定保研方案 (包括推免原则、考核办法、推免程序等内容)，并进行公布。

  2. 学院的程序上可以对学校政策做收紧，但不得放宽。

  3. 普通保研的考核应当只看绩点排名，或者同时计算绩点排名和加分。

  4. 同时计算绩点排名和加分时，绩点比例不得低于 80%，同时加分项目主要包含科研成果、竞赛获奖、参军入伍服兵役、参加志愿服务、到国际组织实习等。各个加分项目有单项上限分值，有同类多次加分的，仅取最高加分项。

加分项目的要求

  1. 学生本科阶段在核心期刊上以独立作者或第一作者发表的与学业相关的科研论文可做加分

  2. 作为主力成员参加与学业相关的国内权威科研竞赛 (全国赛) 并获得三等奖以上奖励可做加分 (国际赛事参照执行，但不得低于国内赛事相关要求) 。

  3. 学生与直系亲属或学历、职称、职务明显高于本人者 (也就是和研究生、讲师、教授等) 合作的科研成果、竞赛奖项等仅作为参考，不纳入加分，但同等条件下可优先考虑。

  4. 学生有加分项目的，需要进行公开答辩。

取消保研的情况

  1. 受到处分

  2. 未能取得学士学位

  3. 毕业论文 (设计) 成绩低于良好

  4. 身体检查不符合录取条件

  5. 在申请推免生过程中弄虚作假

  6. 申请专项计划，毕业时未达到专项计划的要求

  7. 未按规定报备声明回避关系且影响推免过程和结果公平公正

其他补充

  1. 因重修有多个成绩的，成绩按及格后的最低分计算；自 2021 级起，因不及格重修课程后成绩及格的，其课程成绩按 60 分计算。

  2. 普通计划类、专项计划类不能同时申请。

  3. 除有特殊政策要求的专项计划外，所有保研生均可自主选择报考招生单位和专业，所有名额均可向其他招生单位推荐。各学院 (部) 不得阻碍学生自主选择。

  4. 保研的推荐和接收是两个不同阶段。在截至时间前，获得推免资格的学生如果无招生单位接收或本人放弃，则资格作废，名额不得转让。

  5. 体育类、艺术类专业可根据实际情况，适当放宽条件。

本办法由教务处负责解释，最终执行细则以教务处公布为准
