## 贡献指南

1. [注册 GitHub](https://gitHub.com) 账户并登录
1. 在对应页面左下角找到 “编辑此页” 按钮点击跳转
1. 如果从来没有进行过编辑，请点击屏幕中间的绿色按钮 “fork this repository” 进行仓库的复制
1. 在新打开的页面中对内容进行修改，并点击屏幕右下角进行保存。
